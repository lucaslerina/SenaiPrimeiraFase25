DOM é um acrônimo para Document Object Model (Modelo de Objeto de Documento), uma interface de programação que permite que

códigos (como o JavaScript) alterem e interajam com o conteúdo, a estrutura e o estilo de documentos HTML, XML e SVG.
O DOM representa o documento como uma árvore de objetos (nós), onde cada nó é uma parte do documento, e usa métodos para
acessar e modificar essa árvore, permitindo a criação de páginas web dinâmicas e interativas.

Como funciona o DOM:

Representação em Árvore: Quando um navegador carrega uma página, ele a converte em uma estrutura de árvore,
onde o elemento é a raiz, e elementos como <.HEAD>, <.BODY>, e outros elementos dentro deles são seus nós filhos.

Acesso e Manipulação: O DOM fornece um conjunto de métodos e propriedades para que os códigos (como JavaScript) possam acessar
e manipular essa árvore.

Interatividade: Com o DOM, o JavaScript pode:

Alterar o conteúdo: de elementos (como mudar um texto).
Modificar o estilo: da página (como mudar cores ou tamanhos de fontes).
Adicionar ou remover elementos: da página.
Responder a eventos: do usuário (como cliques em botões).

Diferença entre HTML e DOM:
HTML é a linguagem que define a estrutura e o conteúdo inicial de uma página.
DOM é a representação dessa estrutura HTML, transformada em uma árvore de objetos acessíveis e manipuláveis.
